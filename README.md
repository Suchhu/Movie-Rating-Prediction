# Movie Rating Prediction Using Python

## Objective
Predict movie ratings using machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Algorithm
Random Forest Regressor

## Features Used
- Genre
- Director
- Actor 1

## Evaluation Metrics
- Mean Absolute Error (MAE)
- R2 Score

## How to Run

```bash
pip install -r requirements.txt
python movie_rating_prediction.py
```
